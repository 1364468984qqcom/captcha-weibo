## 使用方法
### 下载或克隆整个项目  
```
git clone https://github.com/skygongque/captcha-weibo.git
```

### 安装依赖

```
pip install -r requirements.txt
```

### 启动flask服务

```
python app.py
```

### 测试验证码识别

在`client.py`
